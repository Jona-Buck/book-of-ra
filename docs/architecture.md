# Book of Ra Architecture

## Branches
- main = stable and playable
- ChatGPT = development branch for ChatGPT changes

## Planned Structure
- index.html -> entry point
- css/ -> styles
- js/ -> game logic
- assets/ -> images and sounds

## Workflow
1. Develop on ChatGPT branch
2. Test changes
3. Create Pull Request
4. Merge into main only after testing
